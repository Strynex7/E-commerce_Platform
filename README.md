# E-commerce_Platform

## Description

This is an initial, beginner-level e-commerce platform project created for a client. It serves as a foundation for building a more complex and feature-rich online store. This project is built using HTML, CSS, and JavaScript, focusing on front-end implementation.

## Features

*   **Product Listing**: Displays a catalog of products with key information such as name, image, and price.
*   **Product Details Page**: Provides a detailed view of each product, including descriptions and additional images.
*   **Category Navigation**: Allows users to browse products by category.
*   **Responsive Design**: Ensures the platform is accessible and usable on various devices (desktops, tablets, and mobile phones).
*   **Basic Shopping Cart**: Implements a simple shopping cart functionality to add and remove items.
*   **User Interface**: A clean and intuitive user interface for easy navigation and a pleasant shopping experience.

## Getting Started

### Prerequisites

This project is a front-end application and requires only a modern web browser to run (e.g., Chrome, Firefox, Safari). No additional software or dependencies are required.

### Installation

1.  Clone the repository:

    ```bash
    git clone https://github.com/Strynex7/E-commerce_Platform.git
    ```
2.  Navigate to the project directory:

    ```bash
    cd E-commerce_Platform
    ```

## Usage

Simply open the `index.html` file in your web browser to view the e-commerce platform.

```bash
open index.html
```

Or, if you're on Windows:

```bash
start index.html
```
